This page returns the listing of engine tags which return a variables value. There are other tags such as include, library, tag, tagelse and tagend which can be used to change the contents or the page.

Tag Value Description
User Variables
user:id Returns the users identifier.
user:name Returns the users name.
user:email Returns the users email.
user:phone Returns the users phone number.
Model Variables
model:id Returns the model identifier.
model:name Returns the model name.
Application Variables
application:id Returns the application identifier.
application:name Returns the application name.
application:route Returns the application route.
Screen Variables
screen:name Returns the screen name.
