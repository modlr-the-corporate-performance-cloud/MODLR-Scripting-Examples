MODLR Handlebars Example

Variables

variableOne: {{ variableOne }}
hello: {{ hello }}

The variable below will not format HTML Elements

secure: {{ secure }}

The variable below will format HTML Elements

unsecure: {{{ unsecure }}}

Loops

Loop through array of numbers

arrayNumbers:
{{#each arrayNumbers}} {{this}} {{/each}}

Loop through array of objects

arrayObjects:
{{#each arrayObjects}} {{ name }}, {{ city }}
{{/each}}

Statements

If variable is true

isTrue: {{#if isTrue}} is true {{else}} is false {{/if}}

If variable is false

ifFalse: {{#if ifFalse}} is true {{else}} is false {{/if}}
